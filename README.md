# 📊 Social Media Sentiment Analysis Dashboard

## 📌 Overview
This project analyzes social media text data to understand public sentiment towards different entities.  
The data is cleaned using Python and visualized using Power BI to extract meaningful insights.

---

## 🚀 Tools & Technologies
- Python (Pandas, Regex)
- Jupyter Notebook
- Power BI

---

## 📂 Dataset
- Social media (Twitter-like) dataset  
- Columns:
  - `Id`
  - `Entity`
  - `Sentiment` (Positive, Negative, Neutral, Irrelevant)
  - `Content` (text data)

---

## 🧹 Data Processing
- Converted text to lowercase  
- Removed:
  - URLs  
  - Special characters  
  - Noise  
- Created a new column: `clean_text`  

---

## 📊 Dashboard Features

### 🔹 KPI Metrics
- Total Tweets  
- Positive Tweets  
- Negative Tweets  
- Neutral Tweets  

### 🔹 Visualizations
- Pie Chart → Sentiment Distribution  
- Bar Chart → Entity-wise Tweet Count  

---

## 📈 Key Insights
- Overall sentiment distribution across tweets  
- Identification of most discussed entities  
- Comparison between positive, negative, and neutral sentiments
